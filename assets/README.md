# Assets

This folder contains visual outputs from the regression analysis performed in R Studio.

## Included Visualizations

- `linear-regression-summary.png`  
  R output summary table displaying regression coefficients, standard errors, t-values, and p-values.

- `scatter-linear-fit.png`  
  Scatter plot of flight price vs. distance with fitted linear regression line.

- `quadratic-fit.png`  
  Visualization comparing quadratic regression fit to the linear model.

These visuals are embedded in the main project README for presentation purposes.
